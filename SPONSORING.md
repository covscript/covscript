## Fund Usage

### Fund Allocation

Sponsorship funds will be used according to the following principles:

1. **Project Operations** (40%)
   - Server and infrastructure maintenance
   - Domain names and related tool costs
   - Code hosting service payments
   - Open source community event funding

2. **Contributor Bonuses** (60%)
   - Distributed to active contributors based on contribution metrics
   - Encourages ongoing community participation and high-quality contributions

### Bonus Distribution Mechanism

#### Contribution Assessment

We will comprehensively evaluate contributions across the following dimensions:

- **Code Contributions** (40%)
  - Number and quality of merged PRs
  - Lines of code and complexity
  
- **Issue Resolution** (30%)
  - Number of issues resolved
  - Importance of bug fixes
  
- **Community Service** (20%)
  - Code reviews and feedback
  - Documentation improvements
  - Mentoring newcomers and community support
  
- **Other Contributions** (10%)
  - Promotion and outreach
  - Translation and localization
  - Creative improvement suggestions

#### Distribution Schedule

- **Calculation Period** - Each calendar year
- **Distribution** - Before the end of January of the following year
- **Transparency** - Detailed list published before distribution (contributors can choose whether to make it public)

## Financial Transparency

We are committed to managing sponsorship funds with complete transparency:

### Regular Reports

- Published quarterly in the project's [GitHub Discussions](https://github.com/covscript/covscript/discussions) with financial updates
- Detailed annual financial report published each year

### View Reports

- Check the "Funding & Sponsoring" category in project Discussions
- Detailed financial data will be saved in project documentation

### Privacy Protection

- Sponsors can choose public or anonymous sponsorship
- We will not disclose sponsor information without consent
- Contributors can choose whether to publicly display their bonus amounts

## FAQ

### Q: Does sponsorship need to be a one-time payment?
A: Yes, currently sponsorship is a one-time contribution. If you'd like to be a regular sponsor, you can make a new sponsorship each year.

### Q: Can I sponsor specific feature development?
A: Sponsorship funds support overall project development. If you have specific needs, you're welcome to create an issue on GitHub and let the community vote on priorities.

### Q: Can I sponsor if I'm not from mainland China?
A: Due to legal restrictions, we currently only accept sponsorships from mainland China. However, you can:
- Contribute through code, bug reports, and other means
- Recommend the project to friends in mainland China
- Provide valuable suggestions

### Q: Can I get a refund?
A: Sponsorship is a voluntary goodwill donation and generally cannot be refunded. Please make sure of your decision before sponsoring.

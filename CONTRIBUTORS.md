## How to Contribute

If you'd like to contribute to Covariant Script, please see [CONTRIBUTING.md](CONTRIBUTING.md) for guidelines on:
- Reporting bugs
- Suggesting enhancements
- Submitting code contributions
- Development setup and testing

Your contributions are appreciated and will be recognized in the project history!

## Recognition Policy

- **Code contributions** are attributed via Git commit history
- **Significant contributors** may be mentioned in version release notes
- **Community involvement** helps shape the future of the language

## 如何贡献

如果你想为 Covariant Script 贡献代码，请查看 [CONTRIBUTING-zh.md](CONTRIBUTING-zh.md) 了解以下指南：
- 报告 Bug
- 提出功能建议
- 提交代码贡献
- 开发环境配置和测试

你的贡献将被珍视，并在项目历史中得到认可！

## 认可政策

- **代码贡献** 通过 Git 提交历史自动署名
- **重要贡献者** 可能会在版本发布说明中被提及
- **社区参与** 帮助塑造这门语言的未来

感谢你让 Covariant Script 变得更好！
